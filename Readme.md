## Knowing what is Python?
-  it is a high level programming language used to build all sort of technology and application from software, website to robotics and LLM. It has a plus point of having simpler syntax (english like) making a great choice for begineers

> High Level language means, you donot need to work on hardcore memory management of the computer

> Amazing facts! Python by named by **Guido van Rossum** in 1970s from “Monty Python's Flying Circus”, a BBC comedy series

Python is  super versatile and used in the following areas:
- Data analysis & visualization
- Artificial intelligence (AI)
- Machine learning (ML)
- Web development
- And more!

## Downloading Python

1. Go to the official [Python website](https://www.python.org/downloads/).
2. Choose the appropriate version for your operating system:
   - For Windows: Click on "Download Python [version number]" (e.g., Python 3.x.x).
   - For macOS: You can use the same link to download the installer for macOS.
   - For Linux: Python is often pre-installed, but you can install or update it using your package manager (sudo apt install python).
   - You can verify the installation by opening Command Prompt and typing:
    ```bash
   python --version```
